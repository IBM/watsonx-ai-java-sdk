# Schema Services

Schema Services provide functionality for working with document schemas in `watsonx.ai`. These services enable you to automatically generate, manage, and manipulate structured field definitions from documents, which can then be used for downstream [TextExtraction](../text-extraction-service) and [TextClassification](../text-classification-service) tasks.
